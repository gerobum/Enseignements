Vers le modèle MVC

Un petit exercice pour introduire le modèle MVC.

Un formulaire doit permettre la saisie de quelques informations à propos d'une personne : ses nom, prénom, âge et genre.

La saisie est répétée tant que le formulaire n'est pas correct. Les erreurs de saisie sont indiquées.

En réalité, pour simplifier les choses, un formulaire est correct si l'âge est correct. C'est-à-dire s'il est convertible en entier naturel.
